# RichTextEditor React demo

A React 19 + Vite 8 example showing the RichTextEditor component bridge and self-hosted editor runtime.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The editor assets are served from `public/richtexteditor`. See `src/RichTextEditor.jsx` and `src/richTextEditorBridge.js` for lifecycle and API integration.
